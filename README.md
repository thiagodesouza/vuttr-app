# VUTTR - Very Useful Tools to Remember

This project was created to BossaBox Challenge - Front-end.

<br />

## Download and Run Application

```shell
$ git clone https://github.com/thiagodesouza/vuttr-app.git
$ cd vuttr-app/
$ yarn || npm install
$ yarn start || npm start
```
